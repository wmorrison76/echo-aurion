## Read This First

This supplement adds **six features** to the EchoWaste module. Three are **Phase 1 extensions** to ship alongside or immediately after the v1.0 MVP. Three are **Phase 2 signals** — the schema and API shapes are defined now so Phase 1 implementation choices don't block them, but the actual build waits.

## The Six Features

| # | Feature | Phase | Why |
|---|---|---|---|
| 1 | Buffet pre-cost estimation | 1 extension | Math on existing data — trivial to add |
| 2 | Cost-per-cover calculation | 1 extension | One new input field + math |
| 3 | Plus-sign multi-zone scanning | 1 extension | New table, small UI addition |
| 4 | Refill tracking (prep-line model) | 2 | Requires chef adoption of new habit |
| 5 | Menu pre-load (3 input modes) | 2 | New OCR + extraction pipeline |
| 6 | Verbal recipe creation (draft-only) | 2 | Integrates with culinary module — needs cross-team |
