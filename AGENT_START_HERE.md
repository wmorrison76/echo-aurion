# Agent Start Here

> If you are an AI coding agent and this is the first file you opened in this scaffolding: you are in the right place.

## The 60-second orientation

1. This is a scaffolding for the **Echo Resonance Platform** — an anticipatory hospitality system. Approximately 200 files. None functionally implemented; all are typed stubs.
2. Your job: implement them, in the order specified by **`IMPLEMENTATION_ORDER.md`**.
3. Before you write any code, read **`HANDOFF.md`** — it has the questions you need to ask the human first.
4. Read **`PRIVACY_TENETS.md`** before writing any code that touches guest data. The eight tenets are non-negotiable.
5. Every file has a header that tells you what it is, what it depends on, what is pending. Read the header before implementing.

## The four key documents in priority order

1. `HANDOFF.md` — questions to ask the human, file header conventions, tenet enforcement
2. `IMPLEMENTATION_ORDER.md` — what to build first, second, third
3. `PRIVACY_TENETS.md` — the eight non-negotiable rules
4. `ARCHITECTURE.md` — the navigational map of the whole platform

## Your first action

Open `HANDOFF.md` and identify the **Priority 1 questions** that you need answered before starting. Ask them one at a time, in the order listed. Do not invent tooling choices.

## When in doubt

The scaffolding is the source of truth. The tenets are the constitution. If something is unclear, surface it to the human rather than guessing.

Good luck.
