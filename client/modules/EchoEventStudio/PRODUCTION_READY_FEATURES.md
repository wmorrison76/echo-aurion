## Getting Started

### For First-Time Users
1. Click **Help** button in top toolbar
2. Select **Guided Tour** to learn basics
3. Or jump into the canvas and start designing!

### For Power Users
- Use **Shift+?** to access all keyboard shortcuts
- Press **G** to toggle grid for precise alignment
- Drag panels around to customize your workspace
- Use **Ctrl+S** to save your work frequently

## Panel Management

### Moving Panels
1. Click and drag the panel header (grab handle visible)
2. Panels can be positioned anywhere on screen
3. Panels stay above canvas when dragging (z-index: 1000)

### Collapsing Panels
1. Click the chevron (^) in panel header
2. Panels collapse to header-only view
3. Perfect for decluttering your workspace

### Closing Panels
1. Click the X button in panel header
2. Panels can be restored via Help menu

## Troubleshooting

### Panels not moving?
- Ensure you're clicking the panel header (grab area)
- Check that panel is not locked (look for lock icon)

### Glass effect not visible?
- Update to latest browser version
- Enable hardware acceleration in browser settings
- Check if dark/light mode is properly applied

### Export not working?
- Check browser download permissions
- Verify file size limits (max 50MB)
- Try a different export format

### Help not showing?
- Press `Shift+H` to open help dialog
- Or click Help button in top toolbar
- Use `Shift+?` for keyboard shortcuts
