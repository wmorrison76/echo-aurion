# Transfer Cleanup Checklist

## Critical Exclusions (MUST DELETE)

Before transferring to the other Builder.io instance, ensure the following are **completely removed**:

### 🔴 Dependencies & Lock Files

- [ ] `node_modules/` — **DELETE ENTIRE DIRECTORY**
- [ ] `pnpm-lock.yaml` — **DELETE FILE**
- [ ] `package-lock.json` — **DELETE FILE** (if exists)
- [ ] `yarn.lock` — **DELETE FILE** (if exists)

**Reason**: Host environment manages dependencies; these will cause conflicts and bloat the transfer.

---

### 🔴 Version Control & Metadata

- [ ] `.git/` — **DELETE ENTIRE DIRECTORY**
- [ ] `.gitignore` — **OPTIONAL**: Delete if host has its own
- [ ] `.gitattributes` — **OPTIONAL**: Delete if not needed

**Reason**: Git metadata is source-control only; not needed in the transferred module.

---

### 🔴 Secrets & Environment Files

- [ ] `.env` — **DELETE FILE**
- [ ] `.env.local` — **DELETE FILE**
- [ ] `.env.production` — **DELETE FILE**
- [ ] `.env.development` — **DELETE FILE**
- [ ] `.env.example` — **KEEP** (safe template only)

**Reason**: Never transfer secrets; `.env.example` shows structure without exposing keys.

---

### 🔴 Build & Compiled Artifacts

- [ ] `dist/` — **DELETE ENTIRE DIRECTORY**
- [ ] `build/` — **DELETE ENTIRE DIRECTORY**
- [ ] `.next/` — **DELETE ENTIRE DIRECTORY** (if exists)
- [ ] `out/` — **DELETE ENTIRE DIRECTORY** (if exists)
- [ ] `.vite/` — **DELETE ENTIRE DIRECTORY** (if exists)
- [ ] `*.tsbuildinfo` — **DELETE FILES**

**Reason**: Compiled code is redundant; host will rebuild from source.

---

### 🔴 IDE & Editor Settings

- [ ] `.vscode/` — **DELETE ENTIRE DIRECTORY**
- [ ] `.idea/` — **DELETE ENTIRE DIRECTORY** (IntelliJ)
- [ ] `.DS_Store` — **DELETE FILES** (macOS)
- [ ] `Thumbs.db` — **DELETE FILES** (Windows)
- [ ] `.editorconfig` — **OPTIONAL**: Keep for consistency

**Reason**: Developer-specific settings; host has its own IDE config.

---

### 🔴 Logs & Temporary Files

- [ ] `*.log` — **DELETE ALL LOG FILES**
- [ ] `npm-debug.log*` — **DELETE FILES**
- [ ] `.eslintcache` — **DELETE FILES**
- [ ] `coverage/` — **DELETE DIRECTORY** (if exists)
- [ ] `.nyc_output/` — **DELETE DIRECTORY** (if exists)

**Reason**: Logs are runtime artifacts; not needed for transfer.

---

### 🔴 Package Manager Caches

- [ ] `.pnpm-store/` — **DELETE DIRECTORY** (if exists)
- [ ] `.npm/` — **DELETE DIRECTORY** (if exists)
- [ ] `cache/` — **DELETE DIRECTORY** (if exists related to npm/pnpm)

## Step-by-Step Cleanup

### Option 1: Manual Cleanup (Safest)

```bash
# Navigate to project root
cd echo-event-studio

# Delete each directory carefully
rm -rf node_modules
rm -rf .git
rm -rf dist build
rm -rf .vscode .idea
rm -rf coverage .nyc_output

# Delete environment files
rm -f .env .env.local .env.production .env.development

# Delete cache files
find . -name "*.log" -delete
find . -name ".DS_Store" -delete
find . -name ".eslintcache" -delete

# Verify cleanup
ls -lah | grep -E "node_modules|\.git|dist|\.env"
# Should show nothing
```

### Option 2: Automated Cleanup Script

See `cleanup-for-transfer.sh` for a complete automated solution.

### Option 3: Git-Based Cleanup

```bash
# Create clean checkout
git clone --depth 1 <repo> temp-clean
cd temp-clean
rm -rf .git
# Copy temp-clean to final destination
```

## Pre-Transfer Validation

Run these checks before zipping:

```bash
# 1. Ensure no secrets remain
grep -r "SUPABASE_URL" . --exclude-dir=node_modules
# Should only find: .env.example

# 2. Verify source files exist
test -d client && test -d server && test -f package.json
# Should exit with code 0 (success)

# 3. Check for large files
find . -size +10M -type f
# Should be minimal (images/videos only)

# 4. List final structure
tree -L 2 -I 'node_modules'
```

## Creating the Final Transfer Package

### Compress for Transfer

```bash
# Create tar.gz (recommended for Unix/Linux)
tar --exclude='node_modules' \
    --exclude='.git' \
    --exclude='dist' \
    --exclude='build' \
    --exclude='.env' \
    --exclude='.vscode' \
    --exclude='.idea' \
    -czf EchoEventStudio-transfer.tar.gz .

# Or create zip (recommended for cross-platform)
zip -r EchoEventStudio-transfer.zip . \
    -x "node_modules/*" \
    ".git/*" \
    "dist/*" \
    "build/*" \
    ".env" \
    ".vscode/*" \
    ".idea/*"
```

### Verify Package Size

```bash
# Should be 5-20MB
ls -lh EchoEventStudio-transfer.tar.gz
# or
ls -lh EchoEventStudio-transfer.zip
```
