# 🚀 Quick Reference Guide

## 🎯 Core Features at a Glance

### Panels (Draggable & Collapsible)
| Panel | Location | Purpose |
|-------|----------|---------|
| **Status** | Top-Left | Layout validation & compliance |
| **Statistics** | Top-Center | Object counts & utilization |
| **Suggestions** | Top-Right | AI-powered optimization tips |
| **Export & Save** | Left-Bottom | Export options & file management |

**How to Use:**
- 👉 **Drag**: Click header and drag to reposition
- 📍 **Collapse**: Click `^` to collapse panel
- ❌ **Close**: Click `X` to hide panel

## ⌨️ Essential Keyboard Shortcuts

### Grid & View
| Shortcut | Action | When to Use |
|----------|--------|------------|
| `G` | Toggle grid | Align objects precisely |
| `R` | Reset layout | Start over |

### Export & Save
| Shortcut | Action | When to Use |
|----------|--------|------------|
| `Ctrl+S` | Save JSON | Backup your work |
| `Ctrl+E` | Export PNG | Share presentations |
| `Ctrl+P` | 360° View | Preview panorama |

### Help & Reference
| Shortcut | Action | When to Use |
|----------|--------|------------|
| `Shift+H` | Show Help | Learn features |
| `Shift+?` | Show Shortcuts | Reference all shortcuts |

## 💡 Pro Tips

### Layout Design
1. **Leave pathways** - 4-5 feet minimum clearance
2. **Position buffets** - Away from main seating
3. **Use zones** - Group tables by theme
4. **Check compliance** - Green = ADA compliant

### Panel Management
1. **Customize workspace** - Arrange panels to preference
2. **Collapse when full** - Save screen space
3. **Move to edges** - Keep canvas center clear
4. **Pin favorites** - Keep key panels visible

### Exporting
1. **PNG** - For email/presentations
2. **SVG** - For design editing
3. **JSON** - For importing elsewhere
4. **Check format** - Choose right export type

## 🎬 Getting Started (5 Minutes)

### Step 1: Launch Tour (Recommended)
```
1. Click [Help] button
2. Select "Guided Tour"
3. Follow 8-step introduction
```

### Step 2: Create First Layout
```
1. Click canvas to place tables
2. Drag objects to arrange
3. Check Status panel for issues
4. View suggestions in Suggestions panel
```

### Step 3: Save Your Work
```
1. Press Ctrl+S or click Export panel
2. Select "Export JSON"
3. File downloads automatically
```

### Step 4: Export for Sharing
```
1. Press Ctrl+E or use Export panel
2. Select "Export PNG"
3. Share or print file
```

## 🔧 Troubleshooting

### Panels Not Moving?
```
✓ Check if dragging the header (white grab area)
✓ Panel might be locked (look for lock icon)
✓ Try refreshing page if stuck
```

### Glass Effect Not Showing?
```
✓ Update browser to latest version
✓ Restart browser
✓ Disable browser extensions
✓ Clear cache (Ctrl+Shift+Delete)
```

### Export Not Working?
```
✓ Check popup blockers
✓ Try different export format
✓ Check available disk space
✓ Use latest browser
```

### Help Not Appearing?
```
✓ Press Shift+H to open help
✓ Click Help button in toolbar
✓ Use Shift+? for shortcuts
✓ Refresh page
```
