### Quality Assurance Process

**For Each Term:**

- [ ] **Etymology:** Clear origin and language derivation
- [ ] **Definition:** Authoritative, clear, precise
- [ ] **Context:** Where and how it's used
- [ ] **Related Terms:** Links to connected concepts
- [ ] **Confidence Level:** 0.8-1.0 (must be high)
- [ ] **Mastery Level:** fundamental/intermediate/advanced/expert/master
- [ ] **Sources:** Minimum 1 authoritative source

**Validation Workflow:**

1. Data entry by specialist
2. Review by domain expert (chef, sommelier, etc.)
3. Cross-reference with 2+ authoritative sources
4. Testing with Echo's brain to verify retrievability
5. Final approval by knowledge manager
