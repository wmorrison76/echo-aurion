## 🔧 Technical Specifications

### Frontend Stack
- **Framework**: React 18.3.1
- **Language**: TypeScript 5.9.2
- **Build Tool**: Vite 7.1.2
- **Styling**: Tailwind CSS 3.4.17
- **UI Components**: Radix UI
- **State Management**: React Query
- **Speech APIs**: Web Speech API (native)

### Backend Stack
- **Runtime**: Node.js
- **Framework**: Express.js 5.1.0
- **Language**: TypeScript 5.9.2
- **Database**: PostgreSQL via Supabase
- **Authentication**: JWT (jsonwebtoken 9.1.2)
- **AI**: OpenAI GPT-4o-mini
- **Security**: Helmet 7.1.0
- **HTTP**: Axios 1.7.7

### Database
- **Provider**: Supabase (PostgreSQL)
- **Table**: property_summary
- **Columns**: 8 (id, org_id, outlet_id, labor_cost, revenue, tips, report_date, created_at)
- **Indexes**: 2 (org_id, report_date)

### APIs & Services
- **OpenAI API**: GPT-4o-mini for AI responses
- **Speech Recognition**: Browser Web Speech API
- **Speech Synthesis**: Browser SpeechSynthesisUtterance API
- **Supabase**: PostgreSQL database & auth
