## Builder.io CMS - Getting Started

### No Additional Account Required!

✅ **Works with OpenAI API key alone** (no Builder.io account needed for basic functionality)

**However, to use CMS features:**

1. Get Builder.io API key from [Builder.io Dashboard](https://builder.io)
2. Create content models in Builder.io
3. Use EchoCoder to generate CMS-backed components
4. Content updates happen in Builder.io, code stays synced

### Quick Start Example

```typescript
// Generate a product listing that uses Builder.io CMS
import { generateCMSModuleWithAI } from "@/services/echocoderAI";

const module = await generateCMSModuleWithAI({
  description: "Product listing with images and prices",
  moduleName: "Products",
  useBuilderCMS: true,
  cmsModelId: "product", // From your Builder.io space
});

// Generated component automatically:
// ✅ Fetches products from Builder.io CMS
// ✅ Handles loading/error states
// ✅ Includes TypeScript types
// ✅ Creates custom React hook
// ✅ Is production-ready
```
